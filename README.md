# Machine Learning Lab

A collection of Machine Learning laboratory experiments, implementations, and assignments completed as part of the Machine Learning course. This repository demonstrates the practical application of various machine learning algorithms, data preprocessing techniques, model evaluation methods, and data visualization using Python.

## 📚 Course Objectives

* Understand the fundamentals of Machine Learning.
* Learn data preprocessing and feature engineering.
* Implement supervised and unsupervised learning algorithms.
* Evaluate and compare machine learning models.
* Gain hands-on experience with real-world datasets using Python.

## 🛠️ Technologies Used

* Python 3
* Jupyter Notebook
* NumPy
* Pandas
* Matplotlib
* Scikit-learn



> The folder names may vary depending on the experiments included in this repository.

## 🧪 Experiments

The repository contains implementations of topics such as:

* Data Preprocessing
* Exploratory Data Analysis (EDA)
* Linear Regression
* Logistic Regression
* Decision Trees
* Random Forest
* K-Nearest Neighbors (KNN)
* Naïve Bayes
* Support Vector Machine (SVM)
* K-Means Clustering
* Model Evaluation
* Performance Metrics
* Data Visualization

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/<your-username>/<repository-name>.git
cd <repository-name>
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Launch Jupyter Notebook

```bash
jupyter notebook
```

Open any experiment notebook and run the cells sequentially.
